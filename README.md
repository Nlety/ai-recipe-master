## 项目简介

AI 食谱大师是一款智能菜谱生成器，根据现有食材推荐菜品并提供详细做法。

## 核心功能

- 🥕 **智能推荐**：根据现有食材推荐可做菜品
- 🍳 **详细做法**：分步骤烹饪说明
- 🔥 **营养分析**：热量和营养成分信息
- 🌶️ **口味定制**：清淡/香辣/酸甜/咸香
- 💾 **收藏菜谱**：保存喜欢的食谱

## 技术栈

- HTML/CSS/JavaScript
- Tailwind CSS (CDN)
- OpenAI API
- 阿里云 ESA EdgeKV

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
